Modifying an Array

pop() Remove last element from an array Example 
let array = ['Kitley', 'Wants', 'Better', 'Icecream'];
array.pop();
array;
  ['Kitley', 'Wants', 'Better']
push() Add a new element to the end of an array Example 
let array = ['Kitley', 'Says', 'This', 'Sucks'];
array.pop('Hard');
array;
  ['Kitley', 'Says', 'This', 'Sucks', 'Hard']
shift() Removes the first element of an array Example 
let array = ['Kitley', 'Knits', 'Cute', 'Socks'];
array.shift();
array;
  ['Knits', 'Cute', 'Socks']
unshift() Add a new element to the beginning of an array Example 
let array = ['Kitley', 'Owns', 'This', 'House'];
array.unshift('Captain');
array;
  ['Captain', 'Kirtley', 'Owns', 'This', 'House']
split() Creates a new array from a string divided by a specified value Example 
let truth = 'Kitley Loves Puppy Cuddles';
let array = truth.split(" ");
array;
  ['Kirtley', 'Loves', 'Puppy', 'Cuddles']
join() Combine all elements in an array and return a string Example 
let array = ['Kitley', 'Wants', 'All', 'The', 'Food'];
array.join(" ");
array;
  'Kirtley Wants All The Food'
concat() Join multiple arrays into one Example 
let array1 = ['Kitley', 'Is', 'Amazing'];
let array2 = ['She', 'Is', 'Loved', 'By', 'All'];
let array3 = array1.concat(array2);
array3;
  ['Kitley', 'Is', 'Amazing', 'She', 'Is', 'Loved', 'By', 'All']

Sorting an Array

reverse() Reverse the order of all elements in an arrays Example 
let array = ['Kitley', 'Will', 'Eat', 'Godzilla'];
array.reverse();
array;
  ['Godzilla', 'Eat', 'Will', 'Kirtley']
sort() Sorts all elements in an array alphabetically Example 
let array = ['Kitley', 'Favorite', 'Animal', 'Is', 'Sloths'];
array.sort();
array;
  ['Animal', 'Favorite', 'Is', 'Kitley', 'Sloths']

Positioning of Elements in an Array

indexOf() Returns the position of the first element in an array Example 
let array = ['the', 'duck', 'crossed', 'the', 'road', 'with', 'Kirtley', 'such', 'a', 'cute', 'duck'];
array.indexOf('duck');
  1

Unknown

slice() Removes specified elements of a selected string. Start parameter is required, end parameter is optional Example 
let str = 'Abandoned'
str.slice(7);     // ed
str.slice(-2);    // ned
str.slice(2,5);   // and
str.slice(3,-1);  // ndone
str.slice(0);     // Abandoned
